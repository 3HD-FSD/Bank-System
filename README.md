## 📌 Features

- 🔐 User Authentication System (Login / Logout)
- 👥 User Management
- 🛡️ Role-Based Access Control (Permissions)
- 💰 Account Management
- 💸 Deposit & Withdraw Operations
- 🔄 Money Transfer Between Accounts
- 📊 Transaction Handling
- 💾 Data Persistence using Files
